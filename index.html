This website is made using HTML, CSS, Javascript and PHP, as well as designed on Figma.

    Watchgoods Website

    WatchGoods is a website designed for a luxury watch brand. This project is part of the Human Computer Interaction course. The aim of this website is to be a place where users can get to know about WatchGoods and the product that they sell. This website was made using HTML, CSS, and Javascript.

    Cookability

    Cookability was a prototype app made for Proto-A-Thon competition made using Figma. Cookability is an app carefully and thoughtfully designed for neurodivergent individuals. The focus when creating this prototype app was that this app could be easy to use, interactive, and helps neurodivergent individuals in learning how to cook.

    Souschef

    Souschef is a prototype app, and was made as a part of the Artificial Intelligence course. Souschef is a cooking-help app enhanced with AI features to help user cook and create delicious dishes with limited ingredients that they might have at home. This app could be a solution to those living alone who are looking to save money with food, or people who simply want to learn to cook.

    Lapak Kita

This project provides many features, such as uploading portfolio, trending jobs, job recommendations, and many more to enhance freelancers' and talent seekers' experience in finding what they need. This website was made using HTML, CSS, Javascript, and also designed on Figma.
